beanstalk-deploy-tools
====
> Library for deployment AWS Elastic beanstalk
> This library was inspired by [gulp-beanstalk-dpeloy][https://github.com/a0ly/gulp-beanstalk-deploy].


## Overview
```shell
$) npm i beanstalk-deploy-tools
```

## Support feature
- Deploy application on beanstalk environment
  - archive application files
  - upload archived file
  - deploy uploaded file
  - default exclude list on archived
    - .gitignore content
